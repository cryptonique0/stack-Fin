# Contributing

Thanks for your interest in contributing to Stack-Fin.

## Getting Started

1. Fork the repo and create your branch from `main`.
2. Install dependencies:
   - `npm install`
3. Run checks before opening a PR:
   - `npm run lint`
   - `npm test`

## Coding Guidelines

- Keep changes focused and scoped.
- Add or update tests when you change logic.
- Keep contracts versioned (e.g., `*-v1`, `*-v2`).
- Avoid breaking changes without a migration plan.

## Submitting a Pull Request

- Include a clear summary of changes.
- Link relevant issues or discussions.
- Ensure CI passes.

## Code of Conduct

Be respectful and constructive.
